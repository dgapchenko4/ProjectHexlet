Вернуться

В экзамене по обществознанию есть две части. Первая часть состоит из 16 заданий с кратким ответом. Они имеют форму теста и проверяют общие знания и понимание учащимися предмета. Вторая часть состоит из 9 заданий с развернутым ответом, которые позволяют учащимся продемонстрировать глубину своего понимания тем, которые они разбирали на уроках обществознания, способность к аналитическому мышлению и изложению фактов.

Тесты

С.А. Маркин

Обществознание. Справочник для подготовки к ЕГЭ.

Пособие предназначено как для учащихся средней школы при подготовке к Единому государственному экзамену по курсу обществознания. Книга содержит блоки информации по основным терминам и понятиям курса обществознания. Книга поможет ученикам систематизировать имеющиеся знания и, в случае необходимости, своевременно восполнить обнаружившиеся пробелы.

С.А. Маркин

Обществознание. Справочник для подготовки к ЕГЭ. В предлагаемом издании представлен расширенный курс органической химии для выпускников средних школ и абитуриентов при подготовке к сдаче ЕГЭ. Материал в книге изложен с позиций современных представлений о взаимосвязи между строением и свойствами органических молекул. Книга может быть полезна также для учащихся СПО и вузов нехимического профиля.

О.А. Чернышева

Обществознание. 10-11 классы. Сложный план развернутого ответа. Книга предназначена для эффективной подготовки к ЕГЭ по обществознанию и посвящена выполнению задания высокого уровня сложности — составлению сложного плана развёрнутого ответа по предложенной теме.
